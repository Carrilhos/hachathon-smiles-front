# Smiles travel

Smiles travel é um aplicativo que "gamefica" as milhas em forma de expêriencia e level para o usuário. A ideia é que o app tenha um foco no publico mais jovem, que hoje em dia é um publico que a smiles tem dificuldade em atingir. As formas de gerar experiência geram engajamento nas redes sociais da Smiles, assim divulgando mais ainda a marca.

## Formas de ganhar experiência

* Indicando amigos para a plataforma e ganhar expêriencia por isso
* Compartilhando posts no facebook e Instagram
* Marcando amigos em posts no Facebook e Instagram


## Instalação

Recomendamos que seja usado o `Yarn`

* Necessário ter o app [Expo](https://play.google.com/store/apps/details?id=host.exp.exponent&hl=pt_BR&gl=US)

* Rode os seguintes comandos no terminal

```bash
yarn add global expo-cli
git clone https://github.com/Carrilhos/hachathon-smiles-front.git
cd hackathon-smiles-front
yarn
yarn start
```
